# Chat-Application-Talko
A Chat Application ustilizing WebSockets for real time chatting between users. Built using spring boot.
Designed a secure multi-user chat system with Spring Security, BCrypt password hashing, and session-based authentication, backed by a PostgreSQL database (Supabase) with Spring Data JPA for zero-boilerplate database access; implemented friend request lifecycle (send, accept, reject) and private messaging restricted to accepted friends only.
